DSA PBL Project

Dynamic Delivery Slot Scheduling System

A live dispatch simulator that schedules deliveries using a graph, Dijkstra's shortest path, heap-based priority queue, hash maps, and queue-based slot handling.

Scheduler Rule

Priority first, then nearest feasible driver, free slot, and shortest path ETA
